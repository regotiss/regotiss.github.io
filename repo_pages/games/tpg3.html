  • TypingGame-3

    In this game you have to press key that appears on bubble. Score will update according to key you press. Bubble will disappear and your score will increase by 10 points if you press correct key, else decrease chances by 1 each type. There are particular numbers of chances you will get for each level. If chances become less 0 then GAME OVER.See "About TypingGame3.pdf" provided with source code to get more information.
