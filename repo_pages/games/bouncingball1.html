  • Bouncing Ball Game 1

    Game is Simple created using applet. When applet starts bouncing ball starts to jump. You need to keep ball jumping only on platform using left and right arrow keys. At right top score is displayed. Randomly various small lifeline balls are also introduced. If ball fall down then game is over. You can restart again by clicking on play again option. Audio clips are also added.
